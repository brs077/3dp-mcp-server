# list-models

List all models currently loaded in this session.

## When to use

Use when the user wants to see what models are available, or when you need to confirm a model exists before operating on it.

## Instructions

1. Call `list_models()` with no arguments
2. Present the list with each model's name, bounding box, and volume

## Parameters

None.

## Example

```
list_models()
```
